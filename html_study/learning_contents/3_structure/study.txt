-------------------------------------------------------------------------------------------------------------------
✅ HTML 문서의 구조와 핵심 구성 요소
────────────────────────────────────────────
📌 HTML 문서는 크게 세 부분으로 구성됨
────────────────────────────────────────────
1. <!DOCTYPE html>
    - HTML5 문서 선언
    - 브라우저에게 문서 유형을 알려주는 역할

2. <html>
    - 전체 HTML 문서를 감싸는 루트 요소

3. <head>
    - 문서의 정보(메타데이터), 브라우저/검색엔진 용도

4. <body>
    - 사용자에게 실제로 보이는 콘텐츠 영역

────────────────────────────────────────────
📌 <head> 영역: 브라우저와 검색엔진을 위한 정보
────────────────────────────────────────────
주요 태그와 용도
    <title>
        └─ 브라우저 탭 제목, 검색 결과 제목
    <meta charset="UTF-8">
        └─ 문자 인코딩 설정 (글자 깨짐 방지)
    <meta name="description">
        └─ 검색 결과에 표시될 페이지 설명
    <meta name="viewport">
        └─ 모바일 화면 대응 (반응형 웹)
    <meta property="og:title">
        └─ SNS 공유 시 미리보기 정보 설정 (Open Graph)
    <link rel="stylesheet">
        └─ 외부 CSS 파일 연결
    <script src="...">
        └─ 외부 JS 파일 연결

※ SEO, 반응형 웹, 접근성을 위해 <head> 설정은 매우 중요

────────────────────────────────────────────
📌 <body> 영역: 사용자에게 보여지는 실제 콘텐츠
────────────────────────────────────────────
주요 태그 예시
    <h1> ~ <h6>       : 제목 계층
    <p>               : 단락
    <img>             : 이미지 삽입
    <a>               : 하이퍼링크
    <ul>, <li>        : 목록
    <div>, <section>, <article> : 구조화 및 구획

────────────────────────────────────────────
🌳 DOM Tree 관점에서 보는 HTML 구조
────────────────────────────────────────────
- HTML은 브라우저에 의해 트리 형태(DOM)로 해석됨
- 각 태그는 노드(Node)로 구성되어 계층 구조를 가짐
- JS, CSS, 검색엔진, 스크린리더 등은 이 구조를 기반으로 동작함

예시 구조:
    html
    ├── head
    │     └── title
    └── body
          ├── h1
          └── p
